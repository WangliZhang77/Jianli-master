# 服务器连接问题排查

## 错误信息
```
ECONNREFUSED - 连接被拒绝
```

这意味着**后端服务器（端口 3001）没有运行**。

## 解决步骤

### 1. 检查服务器是否启动

查看运行 `npm run dev` 的终端窗口，应该看到两行输出：
```
[0] 服务器运行在 http://localhost:3001
[1] VITE v5.x.x  ready in xxx ms
```

如果只看到 Vite 的输出，说明后端服务器没有启动。

### 2. 检查后端服务器错误

在终端中查找是否有错误信息，常见错误：
- `Cannot find module 'xxx'` - 依赖未安装
- `EADDRINUSE` - 端口被占用
- `SyntaxError` - 代码语法错误

### 3. 单独启动后端服务器测试

在 VSCode 终端中运行：
```bash
npm run dev:server
```

如果后端服务器启动失败，会显示具体的错误信息。

### 4. 检查依赖是否安装

确保所有依赖都已安装：
```bash
npm install
```

### 5. 检查端口是否被占用

如果端口 3001 被占用，可以：
1. 关闭占用端口的程序
2. 或修改 `.env` 文件中的 `PORT` 值

## 快速修复

1. **停止所有正在运行的服务器**（按 Ctrl+C）
2. **重新安装依赖**：`npm install`
3. **重新启动**：`npm run dev` 或双击 `启动.bat`
