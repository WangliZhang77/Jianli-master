# 安装 DOCX 导出功能依赖

## 问题
如果看到错误：`Failed to resolve import "docx"`，说明需要安装依赖包。

## 解决方法

### 方法 1：在 VSCode 终端中安装（推荐）

1. 打开 VSCode
2. 按 `Ctrl + ~` 打开终端
3. 运行以下命令：

```bash
npm install docx file-saver
```

### 方法 2：使用一键启动脚本

双击 `启动.bat`，脚本会自动检查并安装缺失的依赖。

### 方法 3：手动安装

在项目根目录打开命令行，运行：

```bash
npm install docx file-saver
```

## 安装完成后

重新启动开发服务器：

```bash
npm run dev
```

或者双击 `启动.bat`

## 验证安装

安装成功后，在推荐信页面应该能看到"📄 导出 DOCX"按钮。
